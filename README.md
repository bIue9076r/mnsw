# (M)i(n)e (S)(w)eeper

Command Line Minesweeper

## Options:

### Game options:
	-help:   shows this message
	-easy:   sets game to easy
	-ez:     shorter -easy
	-medium: sets game to medium
	-med:    shorter -medium
	-hard:   sets game to hard

### Variable options:
	-s: set random seed
	-x: set width
	-y: set height
	-r: set chance var
	-d: turn on dark mode
	-l: turn on light mode
	-t: seed refresh rate
	-h: unhide Board
	-e: show endseed
